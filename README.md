Overview: 
This project analyzes how the study location (home vs. school) affects academic productivity. 
Over four weeks, I collected and analyzed data from my Russian homework sessions to identify patterns in performance under controlled conditions.

Methodology:
Collected data over a 4-week period
Each study session lasted 2 hours
Controlled conditions included:
No phone usage
Quiet study environment
Consistent study time (morning before class/work)
Similar sleep duration (~8 hours)

Independent variable:
Study location (Home vs. School)

Dependent variable:
Productivity (% of homework completed)

Productivity was calculated as:
(Number of sections completed ÷ total assigned) × 100. Values were rounded up to the nearest whole number.

Analysis was performed using Python (Jupyter Notebook) and the datascience package.

Visualizations included:
Bar chart → Average productivity by location
Line chart → Productivity trend over time
Separated line charts → Home vs School performance
Scatter plot → Sleep vs productivity

The results show that studying in a structured environment leads to higher productivity. 
Even when controlling for factors such as study duration, sleep, and distractions, study location had a clear and consistent impact on performance.
